# Survey-Calibrated Crop Area Estimation

This repository contains the scripts used in the study "Survey-Calibrated Crop Area Estimation Using Earth Observation".

## Structure
- gee/: Google Earth Engine scripts for feature extraction
- python/: Python notebooks for modelling and analysis
- data/: Information about data sources

## Data sources
- LUCAS 2018 survey (Eurostat)
- Sentinel-1 and Sentinel-2 (Copernicus)
